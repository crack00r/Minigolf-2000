## GitHub CLI Command Reference

### Create Issue

```bash
# Determine next issue number
LAST=$(gh issue list --label "security" --json title -q '.[].title' | \
  grep -oP 'SEC-\K\d+' | sort -n | tail -1)
NEXT=$((${LAST:-0} + 1))

# Create security finding
gh issue create \
  --title "SEC-$NEXT: Title here" \
  --body "## Description\n\nDetails..." \
  --label "finding,ğŸ¤– agent,ğŸ”’ alex,security,P1" \
  --milestone "Sprint 1"

# Add to Security Board (Project #7)
ISSUE_ID=$(gh issue view SEC-$NEXT --json id -q '.id')
gh project item-add 7 --owner crack00r --content-id $ISSUE_ID
```

### Update Issue Status

```bash
# To update status on a project board, you need to:
# 1. Get the project field IDs (Status field)
# 2. Get the project item ID for the issue
# 3. Update the field value

# Example: Move issue to "In Progress"
gh project item-list 6 --owner crack00r --format json | \
  jq '.items[] | select(.content.title | contains("TASK-001"))'
```

### Create Pull Request

```bash
# Create PR and link to issue
```
